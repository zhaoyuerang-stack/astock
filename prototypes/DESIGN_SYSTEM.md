# 设计系统 · 策略审计小程序

> 版本:v3 · 2026-07-04 · 设计师:像素君
> 设计哲学:**克制的金融工具感**——专业但不冷冰冰,易用但不玩具化

---

## 1. 设计原则

| 原则 | 说明 | 反例 |
|---|---|---|
| 数据优先 | 金融数据是主角,装饰是配角 | ❌ 用渐变背景抢数字焦点 |
| 克制配色 | 主色 1 个 + 状态色 3 个,不堆色 | ❌ 五颜六色的 tag |
| 层级靠间距 | 用留白和字重区分层级,不靠色块 | ❌ 用大色块分区 |
| 状态用边框 | PASS/WARN/FAIL 用细边框+小点 | ❌ 整行红底白字 |
| 数字等宽 | 所有金融数字 tabular-nums | ❌ 比例字体数字跳来跳去 |

## 5. 圆角系统

| Token | 值 | 用途 |
|---|---|---|
| `--radius-sm` | 4px | 小元素(tag/badge) |
| `--radius-md` | 8px | 按钮/输入框 |
| `--radius-lg` | 12px | 卡片(默认) |
| `--radius-xl` | 16px | 大卡片/裁决卡 |

**不用 20px+ 大圆角(太玩具感),不用 2px 直角(太硬)。**

## 6. 组件规范

### 气泡(对话)
- 助手:白底 + 0.5px 边框 + radius-lg 12px + 左下角 2px
- 用户:主色填充 + 白字 + radius-lg 12px + 右下角 2px

### 卡片
- 白底 + 0.5px 边框 + radius-lg 12px + padding 16px
- 不用阴影(扁平)

### KPI 卡
- 浅灰底 `--bg-subtle` + 无边框 + radius-md 8px + padding 12px
- 标签 10px 大写灰色 + 数字 16px mono 500

### 状态徽章
- 浅底 + 同色边框 0.5px + radius-sm 4px + padding 2px 8px
- 文字 10px 500

### Gate 行
- 白底 + 底部 0.5px 分割
- 左:6px 圆点(状态色)+ Gate 编号(11px 灰)+ 名称(14px)
- 右:状态文字(12px 500 状态色)

### 按钮
- 主按钮:主色填充 + 白字 + radius-md 8px + padding 12px
- 次按钮:透明 + 0.5px 主色边框 + 主色字
