## 二、 算力板块上行节奏的四个关键阶段

通过分析上述12只算力股的月度累积收益曲线，可以将整个周期的上涨节奏清晰划分为以下四个阶段：

### 1. 第一阶段：首波试探与分化拉升（2024-06 至 2024-12）
* **时间区间**：2024年6月至2024年12月
* **特征**：AI芯片作为核心先导指标率先暴发。
  * **寒武纪 (`688256`)** 一马当先，从 $1.00x$ 迅速拉升至12月的 **$3.19x$**。
  * **海光信息 (`688041`)** 同步上扬至 **$2.03x$**。
  * 相比之下，光模块双雄（**中际旭创**、**新易盛**）在该阶段处于消化筹码和盘整状态，中际旭创从 $1.00x$ 阴跌至年底的 **$0.88x$**。这反映了早期资金极度偏好具备垄断性地位的芯片环节，对光模块的预期存在分化。

### 2. 第二阶段：估值回撤与深度洗盘（2025-01 至 2025-05）
* **时间区间**：2025年1月至2025年5月
* **特征**：板块迎来整体 20%~40% 的深幅回调，消化前期泡沫。
  * 寒武纪在2月达到阶段性高点 $3.56x$ 后迅速回撤至3月的 **$3.02x$**，并在5月落回 **$2.93x$**。
  * 中际旭创在4月跌至最低谷的 **$0.60x$**（即相较起点下跌40%）。
  * 浪潮信息、工业富联等服务器股票也在该阶段见底。此阶段为全周期中最大规模的调整期，考验资金的稳定性。

### 3. 第三阶段：主升浪爆发与普涨（2025-06 至 2025-12）
* **时间区间**：2025年6月至2025年12月
* **特征**：AI产业订单加速落地，板块迎来主升浪，几乎所有算力股都在此阶段实现数倍增长。
  * **寒武纪 (`688256`)** 在2025年8月份经历极限拉升，从6月的 $2.92x$ 暴涨至8月的 **$7.23x$**，奠定了算力绝对总龙头的地位。
  * **光模块双雄** 迎头赶上，展现出极强的弹性：中际旭创从6月的 $1.04x$ 暴涨至12月的 **$4.34x$**；新易盛从 $1.15x$ 上涨至年底的 **$4.15x$**。
  * 工业富联、天孚通信等也在此阶段分别触及约 $2.5x$ 和 $3.0x$ 的高位。

### 4. 第四阶段：板块轮动补涨与终极冲刺（2026-01 至 2026-06）
* **时间区间**：2026年1月至2026年6月
* **特征**：AI算力景气度向产业链上游（存储、光芯片）和二线品种扩散，迎来主升浪的终极冲刺。
  * **长光华芯 (`688048`, 光芯片)** 在5月爆发出惊人弹性，从4月的 $10.26x$ 冲高至5月的 **$11.51x$**（最高峰值曾达 $13.06x$）。
  * **普冉股份 (`688766`, AI存储芯片)** 在1月起飞（从12月的 $1.85x$ 暴拉至1月的 **$4.44x$**），并在6月达到 **$10.20x$**。
  * **协创数据 (`300857`, 算力存储)** 同样在6月收尾期拉升至 **$10.30x$**。
  * 板块双雄中际旭创和新易盛在6月分别以 **$9.30x$** 和 **$9.72x$** 的最高峰值区间收尾，寒武纪收于 **$10.21x$**（峰值 $10.90x$）。

## 三、 算力子版块弹性及轮动特征总结

1. **核心芯片领跑先导**（寒武纪、海光信息）：在板块起动期（第一阶段）最先表现，且全周期回撤幅度相对大势更小，具备强韧性，最终回报均在 4-10 倍级别。
2. **光通讯双雄弹性极大**（中际旭创、新易盛、天孚通信）：在调整期（第二阶段）跌幅最深，但主升期（第三阶段和第四阶段）弹性最强，几乎以近乎垂直的角度完成 9 倍以上的跨越。
3. **硬件设备稳健中军**（工业富联、中科曙光、浪潮信息）：主要承接服务器代工与整机集成，弹性较小，全周期最大涨幅在 $1.8x - 3.1x$ 之间，波动更为平缓。
4. **上游元器件/存储晚发爆发**（普冉股份、长光华芯、协创数据）：前中期表现平平，在周期收尾阶段（第四阶段）由于产能紧缺和技术外溢，出现突发性的爆发（如长光华芯和普冉股份在2026年暴拉至10倍以上），呈现明显的补涨特征。
