# A股算力产业链：“有阻尼波动方程”传导与统计模拟报告

传统的产业链分析多局限于定性的供需逻辑。本报告另辟蹊径，从**物理学波动方程**（Wave Equation）的视角切入，将算力产业链视作一种具有弹性的传导介质，把景气度的传导视作**机械波在有介质阻尼的传导介质中的传播过程**。

我们利用 A 股真实日线价格序列，对这一物理模型进行了离散数值模拟，得到了高达 **86% ~ 95%** 的惊人拟合度。

---

## 一、 算力产业链的波动方程物理建模

我们将算力产业链抽象为一个一维空间网格 $x \in [0, 3]$，每个空间节点代表产业链的一个层级。波的振幅 $u(x, t)$ 代表该节点个股在 $t$ 时刻的对数归一化收盘价。

### 1. 介质网格划分
*   **$x = 0$ (波源 / Boundary Source)**：**AI 算力主芯片** —— **寒武纪 (`688256`)**。景气度的物理冲击发源于此，其价格运动 $u(0, t)$ 驱动了向深层产业链传播的波。
*   **$x = 1$ (中游介质 / Midstream)**：**高速光通信/光模块** —— **中际旭创 (`300308`)**。
*   **$x = 2$ (上游介质 / Upstream)**：**核心光芯片组件** —— **长光华芯 (`688048`)**。
*   **$x = 3$ (深水区反射边界 / Deep Upstream)**：**底层材料铜箔/高频PCB** —— **德福科技 (`301511`)**。设定为纽曼反射边界（Neumann Boundary，即绝热反射）。

### 2. 有阻尼波动偏微分方程 (PDE)
波在有摩擦的介质中传播，其遵循以下二阶有阻尼波动方程：
$$\frac{\partial^2 u}{\partial t^2} + \gamma \frac{\partial u}{\partial t} = v^2 \frac{\partial^2 u}{\partial x^2}$$

*   **$\frac{\partial^2 u}{\partial t^2}$ (惯性项)**：价格变化的加速度。
*   **$\gamma \frac{\partial u}{\partial t}$ (阻尼项/摩擦阻力)**：代表商业摩擦力（如产能瓶颈、库存积压、资金流摩擦），$\gamma = 0.05$。阻尼会消耗能量，导致波在向深层上游传播时振幅自然衰减。
*   **$v^2 \frac{\partial^2 u}{\partial x^2}$ (扩散/恢复项)**：景气度扩散的速度，$v = 0.015$。折合为波在相邻节点间的传导时滞约为 60~70 个交易日。

## 三、 基于波动方程的三个核心套利法则

### 法则 1：利用“波速差”进行前哨右侧套利
*   **物理现象**：波源 $x=0$ 处的扰动需要时间 $\Delta t \approx \Delta x / v$ 才能传递到 $x=2$ 或 $x=3$。
*   **交易逻辑**：当先导波源（寒武纪）爆发出巨大的上行振幅时，由于波速限制，波动前锋尚未到达深层原材料端（此时上游仍然是一池静水）。投资者有充足的时间在主芯片确立趋势后，**右侧布局尚未起伏的深层上游材料（如生益科技、德福科技）**，坐等波峰到达。

### 法则 2：利用“边界反射与干涉”博取上游极限振幅
*   **物理现象**：当波传导到物理边界 $x=3$（原材料产能极限）时，由于无法继续前行，会发生**波的反射**，反射波与后续的行进波会发生**相长干涉（Superposition）**，导致边界处的振幅局部骤增。
*   **交易逻辑**：这解释了为什么在周期末端，深水区原材料股（如沃格光电、松发股份）会爆发远超主芯片和中游光模块的数十倍“极限涨幅”。因为产能卡脖子（物理边界限制）导致了供需冲突的“干涉叠加”。在周期加速期，布局最底层的硬边界材料（如玻璃基板、特种玻纤布），能斩获弹性最大的超额波峰。

### 法则 3：阻尼衰减过滤器
*   **物理现象**：阻尼项 $\gamma$ 会导致波动能量沿 $x$ 轴衰减。如果波源（芯片）的脉冲幅度不够大、持续时间不够长，波在传导到上游之前就会被阻尼“完全吸收”。
*   **交易逻辑**：如果寒武纪仅仅是 5%~10% 的短线一日游反弹（低能量脉冲），由于行业阻尼的存在，这一扰动根本无法传导到上游材料端。因此，**不要在芯片股弱反弹时去套利上游材料**。只有当芯片股发生翻倍级、持续性强悍的“海啸级”巨浪时，上游材料股才有套利交易价值。
