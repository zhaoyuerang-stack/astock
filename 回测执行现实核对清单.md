# 回测执行现实核对清单（残酷执行假设）

> 适用：A 股、日频、微盘/小市值、<500万 实盘前的回测自检。
> 目的：把"回测里活着、实盘里死掉"的执行 gap 提前暴露。
> 判定：**作废级**任一未勾 = 回测无效，结论作废（对齐 CLAUDE.md P0）；**复核级**未勾 = 须人工记录后才可用。
> 成本数值不在本清单写死——唯一权威 = `core/engine.py::CostModel` / `cost_model.md`（R-COST-001）。

---

## A. 可交易状态过滤（point-in-time 宇宙）— 作废级

- [ ] **退市/退市整理期**按 point-in-time 状态处理，持仓中退市股按退市整理期价格清算，**不得静默消失**（忽略退市的微盘回测收益是假的）。
- [ ] **ST/\*ST 状态**按 point-in-time 打标；若纳入，涨跌停按 ±5% 处理，而非 ±10%。
- [ ] **新股上市初期**：上市后一字板/连板期间不可买入；不得在涨停封板日按收盘价建仓。
- [ ] **停牌**：当日停牌不可买卖；持仓中停牌 = 冻结，按停牌前价持有，不可在停牌日调出。
- [ ] **股票池 = 你券商真实可交易池**（剔除你账户实际买不到的标的）。

## B. 涨跌停成交规则 — 作废级（执行 gap 主战场）

- [ ] **买入约束**：信号触发时若已涨停（封板未开）→ 当日不可买入，顺延至下一可成交日。
- [ ] **卖出约束**：若跌停（封板未开）→ 当日不可卖出，顺延，并累计顺延期间损失。
- [ ] **涨跌停幅度按板块**：主板 ±10%、创业板/科创板 ±20%、ST ±5%、新股首日特殊规则。
- [ ] **成交价不用无摩擦收盘价**：用次日开盘价或 VWAP + 滑点，杜绝"按收盘价瞬时成交"。
- [ ] **一字板/T 字板**：全天封板期间该方向不可成交，回测必须识别封板状态而非仅看涨跌幅。

## C. T+1 与时序（防未来函数）— 作废级

- [ ] **T+1**：当日买入不可当日卖出。
- [ ] **信号→成交时序**：T 日收盘后算信号，**T+1 开盘/VWAP 成交**；严禁用 T 日收盘价成交 T 日信号（= 未来函数，R-DATA-003）。
- [ ] **调仓时点单一且明确**，全策略一致，不得逐票临时变更。
- [ ] **基本面/公告/业绩**按**公告日/披露日**对齐，不用报告期末日 ffill（R-DATA-003）。
- [ ] **成分股/可交易状态用历史 point-in-time 名单**，不用当前列表回填。

## D. 价格口径 — 作废级

- [ ] 收益率/动量/技术指标用**复权价**。
- [ ] 估值（PE/PB/PS/PCF）用**不复权价**或与定义一致口径（R-DATA-004），禁止后复权价直接算估值。
- [ ] **成交价用真实价格，不是复权价**（复权价只用于序列计算）。

## E. 成本扣除 — 作废级（数值见 CostModel）

- [ ] **佣金**（双边，含最低收费门槛）。
- [ ] **印花税**（卖出单边，用当前费率）。
- [ ] **过户费**。
- [ ] **微盘买卖价差（spread）**：你的真实主成本。<500万 冲击小，但微盘价差宽——**不得用单一 bps 滑点糊弄**。
- [ ] **成本敏感性**：成本翻倍后策略是否仍达标（高换手必做，R-COST-001 / §10）。
- [ ] 报告中明确成本口径。

## F. 流动性与容量（即使你很小）— 复核级

- [ ] **单票仓位 ≤ 当日成交额的 X%**（设定并记录 X，微盘建议保守）。
- [ ] 不把"涨停日放大的假流动性"当成可买入量。
- [ ] 评估**不可退出场景**：微盘停牌/退市/连续跌停导致的被锁仓。

## G. 尾部与 kill-switch 情景 — 复核级

- [ ] **退市归零损失**已显式建模（不是按最后价平滑卖出）。
- [ ] **连续跌停顺延卖出**的累积损失已建模。
- [ ] **微盘流动性踩踏情景**复盘（如 2024 年初类情景）：策略在该窗口的回撤是否触发你预承诺的 kill-switch、是否在清盘线内。

## H. 防自欺复核 — 作废级

- [ ] 回测可由统一引擎复现（R-BT-001），非每策略各写一套。
- [ ] 全市场口径，非只含存活股/高流动性股（R-DATA-002）。
- [ ] 用 `data_lake` 统一口径，非 `data_full` 旧缓存（R-DATA-001）。
- [ ] 没有任何环节使用未来价格、未来成分、未来可交易状态。

---

### 怎么用
1. 每条回测前逐项勾。
2. **任一"作废级"未勾 → 这次回测结论作废，先修再跑**，不得作为有效性证据。
3. "复核级"未勾 → 人工记录原因与影响后方可使用。
4. 微盘策略：A、B、G 三组是你实盘 gap 与暴死风险的命门，重点核。

### 已知边界（诚实）
- 费率/涨跌停幅度/打新经济性等参数会变，以你 CostModel 与交易所现行规则为准，本清单不写死数值。
- 本清单只覆盖**执行现实**层；防未来函数的机械审计仍以 `phase1_synthetic` 为唯一执行点。
